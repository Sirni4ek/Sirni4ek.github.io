Common Command Line Commands

Welcome! Below is a collection of useful command line commands that you might find helpful in your programming journey.

cd

Change directory:

cd ~      # Home directory
cd .      # Current directory
cd ..     # Parent directory

pwd

Print working directory:

pwd

sh

Shell interpreter:

sh

mc

Midnight Commander:

F4      # Edit file
F3      # View file
quickview

ls

List directory contents:

ls *.txt   # List all .txt files
ls -l      # Detailed list
ls -lh     # Human-readable sizes
ls -la     # Include hidden files (.dotfiles)

cat

Catenate (print or output) file contents:

cat filename.txt

STDIN/STDOUT

Standard input/output:

# Standard in (input)
# Standard out (output)

sha1sum

Get the sha1 hash of a file or input:

sha1sum filename.txt

find

Find files in the filesystem:

find -type f    # List only files
find -type d    # List only directories
find -name "*.txt"  # Find files by name

grep

Search for text in files or STDIN:

grep "pattern" filename.txt
^  # Beginning of line
$  # End of line

wc

Word count / line count:

wc filename.txt

less

View large output one screen at a time:

less filename.txt
# Controls:
space    # Next page
page up  # Page up
page down  # Page down
q       # Quit

echo

Print something to the terminal:

echo "Hello, world!"

ps

Process list:

ps aux

git

Version control system:

git status
git add .
git commit -m "Your message"

hexdump

Display file contents in hexadecimal:

hexdump filename.bin

cut

Split output into fields using a delimiter:

cut -d ',' -f 2- filename.csv
# -d ','  # Delimiter is comma
# -f 2-   # Fields we want (starting from 2)

xargs

Use STDOUT as arguments for another command:

echo "file1 file2 file3" | xargs rm

diff

Compare two files:

diff file1.txt file2.txt

tr

Translate input to a different format:

echo "Hello" | tr 'a-z' 'A-Z'

awk

Process text data, search patterns, perform actions:

awk '{print $1}' filename.txt

time

Track how long it takes to run a command:

time command
